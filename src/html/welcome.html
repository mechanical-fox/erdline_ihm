Erdline est un site vitrine, qui a pour but de permettre de générer des documentations API depuis un site web. Et cela de façon intuitive, en remplissant des menus. Et non donc en remplissant un code OpenAI dont la syntaxe sera à apprendre. Le concept de site vitrine fait référence au fait que le but de ce site est principalement de me servir en portfolio, et en CV, afin de pouvoir montrer mes compétences à des employeurs potentiels.


     Cela explique pourquoi actuellement j'ai effectué une première version de ce site, et cela alors que l'on peut générer des documentations API, mais pas encore les sauvegarder pour les reprendre plus tard.

Evolutions futures prévues

1. Pouvoir exporter et importer avec un format spécifique à Erdline (= Sauvegarde)

2. Exporter une documentation API format OpenAI

3. Pouvoir éditer un endpoint déjà crée

4. Permettre l'ajout de description (Description API, endpoint, paramètres)
