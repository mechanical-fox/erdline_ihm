
# Projet   

Le but de ce projet est de réaliser une interface graphique, afin de se connecter à l'API   
Erdline. Le but du site internet sera de gérer les éléments suivant.    

- Créer une documentation API depuis une interface web
- Génération du code OpenAI associé
- Génération de la documentation API en format Html
- Capacité à importer une documentation au format OpenAI

Ce projet est lié a l'API Erdline.   

Ce site est en cours de construction, il est décrit ici le but futur.   
L'API est aussi en cours de construction.
